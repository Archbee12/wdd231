Site Name

BYUI Cloned

Site Purpose

The purpose of this project is to design and build a personalized, student-centered portal inspired by the official BYU-Idaho portal. This site will serve as both a functional homepage and a course documentation tool, providing an organized and visually familiar interface that mimics the original BYUI student experience.

Project Overview

This project will replicate key design elements of the BYUI portal to create a customized student dashboard. The homepage will feature navigation links, a welcome section, and a layout similar to the official portal. In addition to the visual clone, this site will include a Courses section where I will document all the courses I am taking. Each course will display details such as the course name, description, credits. Users will be able to click on course cards to view more information in a modal dialog.

Planned Enhancement

To improve functionality and interactivity, I will be extending the class-schedule.html page to include:

✅ Add/Drop Course Feature: Students will be able to add new courses from a selection list or remove previously added courses from their schedule. This will simulate a basic course registration system.

✅ Local Storage Integration: Selected courses will be stored in the browser’s localStorage to maintain user choices across sessions.

✅ User Feedback: Buttons will update in real-time (e.g., "Add" → "Added", disabled after click) to improve UX.

The site will be fully responsive, accessible, and structured using semantic HTML, CSS, and JavaScript. I plan to implement features such as:

  • A dynamic homepage layout inspired by BYUI's portal design.
  • A course documentation area with interactive modals.
  • JavaScript functionality for fetching and displaying course data from a JSON file.
  • Modern front-end practices like modular JavaScript, local storage, and DOM manipulation.

This project not only demonstrates my web development skills but also serves as a personal academic organizer.

The primary target audience for this project includes:

  • Students – especially BYU-Idaho students or students from other institutions who want a personalized way to track and manage their coursework. The familiar layout helps with usability and quick navigation.

  • Educators & Instructors – instructors seeking student portfolio examples or inspiration for teaching responsive web design, JavaScript modals, and JSON-based data handling.
